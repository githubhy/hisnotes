# 志第四十八 艺文二

1. <span id="志第四十八_艺文二-1"></span>
乙部史录，其类十三：一曰正史类，二曰编年类，三曰伪史类，四曰杂史类，五曰起居注类，六曰故事类，七曰职官类，八曰杂传记类，九曰仪注类，十曰刑法类，十一曰目录类，十二曰谱牒类，十三曰地理类。凡著录五百七十一家，八百五十七部，一万六千八百七十四卷；不著录三百五十八家，一万二千三百二十七卷。
