# 志第二十 礼乐三

## 郊祀乐章

1. <span id="志第二十_礼乐三-郊祀乐章-1"></span>
成宗大德六年，合祭天地五方帝乐章：

2. <span id="志第二十_礼乐三-郊祀乐章-2"></span>
降神，奏《乾宁之曲》，六成：

3. <span id="志第二十_礼乐三-郊祀乐章-3"></span>
圜钟宫三成

4. <span id="志第二十_礼乐三-郊祀乐章-4"></span>
惟皇上帝，监德昭明。祀考承天，治底隆平。孝思维则，禋祀荐诚。神其降格，万福来并。

5. <span id="志第二十_礼乐三-郊祀乐章-5"></span>
黄钟角一成词同前。

6. <span id="志第二十_礼乐三-郊祀乐章-6"></span>
太簇徵一成词同前。

7. <span id="志第二十_礼乐三-郊祀乐章-7"></span>
姑洗羽一成词同前。

8. <span id="志第二十_礼乐三-郊祀乐章-8"></span>
初献盥洗，奏《肃宁之曲》：

9. <span id="志第二十_礼乐三-郊祀乐章-9"></span>
黄钟宫

10. <span id="志第二十_礼乐三-郊祀乐章-10"></span>
明水在下，钟鼓既奏。有孚颙若，陟降左右。辟公处之，多士稞将。吉蠲以祭，上帝其飨。

11. <span id="志第二十_礼乐三-郊祀乐章-11"></span>
初献升降，奏《肃宁之曲》：

12. <span id="志第二十_礼乐三-郊祀乐章-12"></span>
大吕宫

13. <span id="志第二十_礼乐三-郊祀乐章-13"></span>
禋祀孔肃，盥荐初升。摄齐恭敬，以荐惟馨。肃雝多士，来格百灵。降福受厘，万世其承。

14. <span id="志第二十_礼乐三-郊祀乐章-14"></span>
奠玉币，奏：

15. <span id="志第二十_礼乐三-郊祀乐章-15"></span>
大吕宫

16. <span id="志第二十_礼乐三-郊祀乐章-16"></span>
宗祀配飨，肇举明禋。嘉玉既设，量币斯陈。惟德格天，惟诚感神。于万斯年，休命用申。

17. <span id="志第二十_礼乐三-郊祀乐章-17"></span>
迎俎，奏《丰宁之曲》：

18. <span id="志第二十_礼乐三-郊祀乐章-18"></span>
黄钟宫

19. <span id="志第二十_礼乐三-郊祀乐章-19"></span>
有硕斯俎，有涤斯牲。銮刀屡奏，血膋载升。礼崇茧栗，气达尚腥。上帝临止，享于克诚。

20. <span id="志第二十_礼乐三-郊祀乐章-20"></span>
酌献，奏《嘉宁之曲》：

21. <span id="志第二十_礼乐三-郊祀乐章-21"></span>
大吕宫

22. <span id="志第二十_礼乐三-郊祀乐章-22"></span>
崇崇泰畤，穆穆昊穹。神之格思，祇蚃斯通。牺尊载列，黄流在中。酒既和止，万福攸同。

23. <span id="志第二十_礼乐三-郊祀乐章-23"></span>
亚献，奏《咸宁之曲》：

24. <span id="志第二十_礼乐三-郊祀乐章-24"></span>
黄钟宫

25. <span id="志第二十_礼乐三-郊祀乐章-25"></span>
六成既阕，三献云终。神其醉止，穆穆雍雍。和风庆云，贲我郊宫。受兹祉福，亿载无穷。

26. <span id="志第二十_礼乐三-郊祀乐章-26"></span>
终献词同前。

27. <span id="志第二十_礼乐三-郊祀乐章-27"></span>
彻笾豆，奏《丰宁之曲》：

28. <span id="志第二十_礼乐三-郊祀乐章-28"></span>
大吕宫

29. <span id="志第二十_礼乐三-郊祀乐章-29"></span>
禋礼既备，神具宴娭。笾豆有楚，废彻不迟。多士骏奔，乐且有仪。乃锡纯嘏，永佐丕基。

30. <span id="志第二十_礼乐三-郊祀乐章-30"></span>
送神奏：

31. <span id="志第二十_礼乐三-郊祀乐章-31"></span>
圜钟宫

32. <span id="志第二十_礼乐三-郊祀乐章-32"></span>
殷祀既毕，灵驭载旋。礼洽和应，降福自天。动植咸若，阴阳不愆。明明天子，亿万斯年。

33. <span id="志第二十_礼乐三-郊祀乐章-33"></span>
望燎奏：

34. <span id="志第二十_礼乐三-郊祀乐章-34"></span>
黄钟宫

35. <span id="志第二十_礼乐三-郊祀乐章-35"></span>
享申百礼，庆洽百灵。奠玉高坛，燔柴广庭。祥光达曙，灿若景星。神之降福，万国咸宁。

36. <span id="志第二十_礼乐三-郊祀乐章-36"></span>
大德九年以后，定拟亲祀乐章：

37. <span id="志第二十_礼乐三-郊祀乐章-37"></span>
皇帝入中壝：

38. <span id="志第二十_礼乐三-郊祀乐章-38"></span>
黄钟宫

39. <span id="志第二十_礼乐三-郊祀乐章-39"></span>
赫赫有临，洋洋在上。克配皇祖，于穆来飨。肇此大禋，乾文弘朗。被衮圜丘，巍巍玄象。

40. <span id="志第二十_礼乐三-郊祀乐章-40"></span>
皇帝盥洗：

41. <span id="志第二十_礼乐三-郊祀乐章-41"></span>
黄钟宫

42. <span id="志第二十_礼乐三-郊祀乐章-42"></span>
翼翼孝思，明德洽礼。功格玄穹，有光帝始。著我精诚，洁兹荐洗。币玉攸奠，永集嘉祉。

43. <span id="志第二十_礼乐三-郊祀乐章-43"></span>
皇帝升坛：降同。

44. <span id="志第二十_礼乐三-郊祀乐章-44"></span>
大吕宫

45. <span id="志第二十_礼乐三-郊祀乐章-45"></span>
天行惟健，盛德御天。日月龙章，笋虡宫县。藁鞂尚明，礼璧苍圜。神之格思，香升燔烟。

46. <span id="志第二十_礼乐三-郊祀乐章-46"></span>
降神，奏《天成之曲》：

47. <span id="志第二十_礼乐三-郊祀乐章-47"></span>
圜钟宫三成

48. <span id="志第二十_礼乐三-郊祀乐章-48"></span>
烝哉皇元，丕承帝眷。报本贵诚，于郊殷荐。藁鞂载陈，云门六变。神之格思，来处来燕。

49. <span id="志第二十_礼乐三-郊祀乐章-49"></span>
黄钟角一成

50. <span id="志第二十_礼乐三-郊祀乐章-50"></span>
大簇徵一成

51. <span id="志第二十_礼乐三-郊祀乐章-51"></span>
姑洗羽一成词并同前。

52. <span id="志第二十_礼乐三-郊祀乐章-52"></span>
初献盥洗，奏《隆成之曲》：

53. <span id="志第二十_礼乐三-郊祀乐章-53"></span>
黄钟宫

54. <span id="志第二十_礼乐三-郊祀乐章-54"></span>
肇禋南郊，百神受职。齐洁惟先，匪馨于稷。乃沃乃盥，祠坛是陟。上帝监观，其仪不忒。

55. <span id="志第二十_礼乐三-郊祀乐章-55"></span>
初献升坛，降同。 奏《隆成之曲》：

56. <span id="志第二十_礼乐三-郊祀乐章-56"></span>
大吕宫

57. <span id="志第二十_礼乐三-郊祀乐章-57"></span>
于穆圜坛，阳郊奠位。孔惠孔时，吉蠲为饎。降登祗若，百礼既至。愿言居歆，允集熙事。

58. <span id="志第二十_礼乐三-郊祀乐章-58"></span>
奠玉币，正配位同。 奏《钦成之曲》：

59. <span id="志第二十_礼乐三-郊祀乐章-59"></span>
黄钟宫

60. <span id="志第二十_礼乐三-郊祀乐章-60"></span>
谓天盖高，至诚则格。克祀克禋，骏奔百辟。制币斯陈，植以苍璧。神其降康，俾我来益。

61. <span id="志第二十_礼乐三-郊祀乐章-61"></span>
司徒捧俎，奏《宁成之曲》：

62. <span id="志第二十_礼乐三-郊祀乐章-62"></span>
黄钟宫

63. <span id="志第二十_礼乐三-郊祀乐章-63"></span>
我牲既洁，我俎斯实。笙镛克谐，笾豆有飶。神来宴娭，歆兹明德。永锡繁禧，如几如式。

64. <span id="志第二十_礼乐三-郊祀乐章-64"></span>
昊天上帝位酌献，奏《明成之曲》：

65. <span id="志第二十_礼乐三-郊祀乐章-65"></span>
黄钟宫

66. <span id="志第二十_礼乐三-郊祀乐章-66"></span>
于昭昊天，临下有赫。陶匏荐诚，声闻在德。酌言献之，上灵是格。降福孔偕，时万时亿。

67. <span id="志第二十_礼乐三-郊祀乐章-67"></span>
皇地祗位酌献：

68. <span id="志第二十_礼乐三-郊祀乐章-68"></span>
大吕宫

69. <span id="志第二十_礼乐三-郊祀乐章-69"></span>
至哉坤元，与天同德。函育群生，玄功莫测。合飨圜坛，旧典时式。申锡无疆，聿宁皇国。

70. <span id="志第二十_礼乐三-郊祀乐章-70"></span>
太祖位酌献：

71. <span id="志第二十_礼乐三-郊祀乐章-71"></span>
黄钟宫

72. <span id="志第二十_礼乐三-郊祀乐章-72"></span>
礼大报本，郊定天位。皇皇神祖，反始克配。至德难名，玄功宏济。帝典式敷，率育攸塈。

73. <span id="志第二十_礼乐三-郊祀乐章-73"></span>
皇帝饮福：

74. <span id="志第二十_礼乐三-郊祀乐章-74"></span>
大吕宫

75. <span id="志第二十_礼乐三-郊祀乐章-75"></span>
特牲享诚，备物循质。上帝居歆，百神受职。皇武昭宣，孝祀芬苾。万福攸同，下民阴骘。

76. <span id="志第二十_礼乐三-郊祀乐章-76"></span>
皇帝出入小次：

77. <span id="志第二十_礼乐三-郊祀乐章-77"></span>
黄钟宫

78. <span id="志第二十_礼乐三-郊祀乐章-78"></span>
惟天为大，惟帝飨帝。以配祖考，肃赞灵祉。定极崇功，永我昭事。升中于天，象物毕至。

79. <span id="志第二十_礼乐三-郊祀乐章-79"></span>
文舞退，武舞进，奏《和成之曲》：

80. <span id="志第二十_礼乐三-郊祀乐章-80"></span>
黄钟宫

81. <span id="志第二十_礼乐三-郊祀乐章-81"></span>
羽籥既竣，载扬玉戚。一弛一张，匪舒匪棘。八音克谐，万舞有奕。永观厥成，纯嘏是锡。

82. <span id="志第二十_礼乐三-郊祀乐章-82"></span>
亚终献，奏《和成之曲》：

83. <span id="志第二十_礼乐三-郊祀乐章-83"></span>
黄钟宫

84. <span id="志第二十_礼乐三-郊祀乐章-84"></span>
有严郊禋，恭陈币玉。大糦是承，载祗载肃。上帝居歆，馨香既饫。惠我无疆，介以景福。

85. <span id="志第二十_礼乐三-郊祀乐章-85"></span>
彻笾豆，奏《宁成之曲》：

86. <span id="志第二十_礼乐三-郊祀乐章-86"></span>
大吕宫

87. <span id="志第二十_礼乐三-郊祀乐章-87"></span>
三献攸终，六乐斯遍。既右享之，彻其有践。洋洋在上，默默灵眷。明禋告成，于皇锡羡。

88. <span id="志第二十_礼乐三-郊祀乐章-88"></span>
送神，奏《天成之曲》：

89. <span id="志第二十_礼乐三-郊祀乐章-89"></span>
圜钟宫

90. <span id="志第二十_礼乐三-郊祀乐章-90"></span>
神之来歆，如在左右。神保聿归，灵斿先后。恢恢上圆，无声无臭。日监孔昭，思皇多祐。

91. <span id="志第二十_礼乐三-郊祀乐章-91"></span>
望燎，奏《隆成之曲》：

92. <span id="志第二十_礼乐三-郊祀乐章-92"></span>
黄钟宫

93. <span id="志第二十_礼乐三-郊祀乐章-93"></span>
熙事备成，礼文郁郁。紫烟聿升，灵光下烛。神人乐康，永膺戩谷。祚我丕平，景命有仆。

94. <span id="志第二十_礼乐三-郊祀乐章-94"></span>
皇帝出中壝：

95. <span id="志第二十_礼乐三-郊祀乐章-95"></span>
黄钟宫

96. <span id="志第二十_礼乐三-郊祀乐章-96"></span>
泰坛承光，寥廓玄暧。暢我扬明，飨仪惟大。九服敬宣，声教无外。皇拜天祐，照临斯届。

97. <span id="志第二十_礼乐三-郊祀乐章-97"></span>
宗庙乐章

98. <span id="志第二十_礼乐三-郊祀乐章-98"></span>
世祖中统四年至至元三年，七室乐章：《太常集礼藁》云，此系卷牍所载。

99. <span id="志第二十_礼乐三-郊祀乐章-99"></span>
太祖第一室：

100. <span id="志第二十_礼乐三-郊祀乐章-100"></span>
天垂灵顾，地献中方。帝力所拓，神武莫当。阳溪昧谷，咸服要荒。昭孝明禋，神祖皇皇。

101. <span id="志第二十_礼乐三-郊祀乐章-101"></span>
太宗第二室：

102. <span id="志第二十_礼乐三-郊祀乐章-102"></span>
和林胜域，天邑地宫。四方宾贡，南北来同。百司分置，胄教肇崇。润色祖业，德仰神宗。

103. <span id="志第二十_礼乐三-郊祀乐章-103"></span>
睿宗第三室：

104. <span id="志第二十_礼乐三-郊祀乐章-104"></span>
珍符默授，畴昔自天。爰生圣武，宝祚开先。霓旌回狩，龙驾游仙。追远如生，皇慕颙然。

105. <span id="志第二十_礼乐三-郊祀乐章-105"></span>
皇伯考术赤第四室：

106. <span id="志第二十_礼乐三-郊祀乐章-106"></span>
威武鹰扬，冢位克当。从龙远拓，千万里疆。诞总虎旅，驻压西方。航海梯山，东西来王。

107. <span id="志第二十_礼乐三-郊祀乐章-107"></span>
皇伯考察合带第五室：

108. <span id="志第二十_礼乐三-郊祀乐章-108"></span>
雄武军威，滋多历年。深谋远略，协赞惟专。流沙西域，饯日东边。百国畏服，英声赫然。

109. <span id="志第二十_礼乐三-郊祀乐章-109"></span>
定宗第六室：

110. <span id="志第二十_礼乐三-郊祀乐章-110"></span>
三朝承休，恭己优游。钦绳祖武，其德聿修。帝ＭＥ锡寿，德泽期周。蠲饣喜惟芗，祈飨于幽。

111. <span id="志第二十_礼乐三-郊祀乐章-111"></span>
宪宗第七室：

112. <span id="志第二十_礼乐三-郊祀乐章-112"></span>
龙跃潜居，风云会通。知民病苦，轸念宸衷。夔门之旅，继志图功。俎豆敬祭，华仪孔隆。

113. <span id="志第二十_礼乐三-郊祀乐章-113"></span>
至元四年至十七年，八室乐章：《太常集礼》云，周驭所藏《仪注》所录舞节同。

114. <span id="志第二十_礼乐三-郊祀乐章-114"></span>
迎神，奏《来成之曲》，九成：

115. <span id="志第二十_礼乐三-郊祀乐章-115"></span>
黄钟宫三成

116. <span id="志第二十_礼乐三-郊祀乐章-116"></span>
齐明盛服，翼翼灵眷。礼备多仪，乐成九变。烝烝孝心，若闻且见。肸蚃端临，来宁来燕。

117. <span id="志第二十_礼乐三-郊祀乐章-117"></span>
大吕角二成词同黄钟。

118. <span id="志第二十_礼乐三-郊祀乐章-118"></span>
大簇徵二成词同黄钟。

119. <span id="志第二十_礼乐三-郊祀乐章-119"></span>
应钟羽二成词同黄钟。

120. <span id="志第二十_礼乐三-郊祀乐章-120"></span>
初献盥洗，奏《肃成之曲》：再诣盥洗同。至大以后，名《顺成之曲》，词律同。

121. <span id="志第二十_礼乐三-郊祀乐章-121"></span>
无射宫

122. <span id="志第二十_礼乐三-郊祀乐章-122"></span>
天德维何，如水之清。维水内耀，配彼天明。以涤以濯，牺象光晶。孝思维则，式荐忱诚。

123. <span id="志第二十_礼乐三-郊祀乐章-123"></span>
初献升殿，登歌乐奏《肃成之曲》：降同。

124. <span id="志第二十_礼乐三-郊祀乐章-124"></span>
夹钟宫

心传忠恕，一以贯之。爰述《大学》，万工训彝。惠我光明，尊闻行知。继圣迪后，是享是宜。

363. <span id="志第二十_礼乐三-郊祀乐章-363"></span>

公传自曾，孟传自公。有嫡绪承，允得其宗。提纲开蕴，乃作《中庸》。侑于元圣，亿载是崇。

366. <span id="志第二十_礼乐三-郊祀乐章-366"></span>

南吕宫

道之由兴，于皇宣圣。维公之传，人知趋正。与飨在堂，情文斯称。万年承休，假哉天命。

369. <span id="志第二十_礼乐三-郊祀乐章-369"></span>

姑洗宫

百王宗师，生民物轨。瞻之洋洋，神其宁止。酌彼金罍，惟清且旨。登献惟三，于嘻成礼。

372. <span id="志第二十_礼乐三-郊祀乐章-372"></span>
饮福受胙。与盥洗同，惟国学释奠亲祀用之，摄事则不用，外路州县并皆用之。

南吕宫

牺象在前，豆笾在列。以享以荐，既芬既洁。礼成乐备，人和神悦。祭则受福，率尊无越。

黄钟宫

379. <span id="志第二十_礼乐三-郊祀乐章-379"></span>
望瘗。与盥洗同。

右释奠乐章，皆旧曲。元朝尝拟撰易，而未及用，今并附于此。

381. <span id="志第二十_礼乐三-郊祀乐章-381"></span>
迎神，奏《文明之曲》：

天纵之圣，集厥大成。立言垂教，万世准程。庙庭孔硕，尊俎既盈。神之格思，景福来并。

383. <span id="志第二十_礼乐三-郊祀乐章-383"></span>

385. <span id="志第二十_礼乐三-郊祀乐章-385"></span>

大哉圣功，薄海内外。礼隆秩宗，光垂昭代。陟降在庭，摄齐委佩。莫不肃雝，洋洋如在。

387. <span id="志第二十_礼乐三-郊祀乐章-387"></span>
奠币，奏《德明之曲》：

圭衮尊崇，佩绅列侑。笾豆有楚，业具和奏。式陈量币，骏奔左右。天睠斯文，繄神之佑。
