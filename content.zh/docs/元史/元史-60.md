# 志第十二 地理三

## 陕西诸道行御史台

1. <span id="志第十二_地理三-陕西诸道行御史台-1"></span>
陕西等处行中书省，为路四、府五、州二十七，属州十二，属县八十八。本省陆站八十处，水站一处。

2. <span id="志第十二_地理三-陕西诸道行御史台-2"></span>
奉元路，上。 唐初为雍州，后改关内道，又改京兆府，又以京城为西京，又曰中京，又改上都。宋分陕西永兴、秦凤、熙河、泾原、环庆、鄜延为六路。金并陕西为四路。元中统三年，立陕西四川行省，治京兆。至元初，并云阳县入泾阳，栎阳县入临潼，终南县入盩啡。十六年，改京兆为安西路总管府。二十三年，四川置行省，改此省为陕西等处行中书省。大德元年，移云南行台于此，为陕西行台。

3. <span id="志第十二_地理三-陕西诸道行御史台-3"></span>
皇庆元年，改安西为奉元路。户三万三千九百三十五，口二十七万一千三百九十九。

4. <span id="志第十二_地理三-陕西诸道行御史台-4"></span>
壬子年数。 领司一、县十一、州五。州领十五县。

5. <span id="志第十二_地理三-陕西诸道行御史台-5"></span>
录事司。

6. <span id="志第十二_地理三-陕西诸道行御史台-6"></span>
县十一

7. <span id="志第十二_地理三-陕西诸道行御史台-7"></span>
咸宁，下。 长安，下。 咸阳，下。 兴平，下。 临潼，下。屯田一千二十顷有奇。 蓝田，下。 泾阳，下。至元二年，并入高陵县。三年复立。屯田一千三十顷有奇。 高陵，下。 鄠县，下。

8. <span id="志第十二_地理三-陕西诸道行御史台-8"></span>
盩啡，下。屯田九百四十三顷有奇。 郿县。下。旧为郿州，添置柿林县。至元元年，省郿州为郿县，废柿林。

9. <span id="志第十二_地理三-陕西诸道行御史台-9"></span>
州五

10. <span id="志第十二_地理三-陕西诸道行御史台-10"></span>
同州，下。 唐初为同州，又改冯翊郡，又复为同州。宋为定国军。金因之。

11. <span id="志第十二_地理三-陕西诸道行御史台-11"></span>
元仍为同州。领五县：

12. <span id="志第十二_地理三-陕西诸道行御史台-12"></span>
朝邑，下。 白水，下。 郃阳，下。 澄城，下。 韩城。下。唐、宋为韩城县，金曰桢州。至元元年，州废。二年再立。六年，州又废，止设县。

13. <span id="志第十二_地理三-陕西诸道行御史台-13"></span>
华州，下。 唐改镇国军。宋改镇潼军。金改金安军。元复为华州。西岳华山在焉。 领三县：

14. <span id="志第十二_地理三-陕西诸道行御史台-14"></span>
华阴，下。 蒲城，下。 渭南。下。屯田一千二百二十二顷有奇。

15. <span id="志第十二_地理三-陕西诸道行御史台-15"></span>
耀州，下。 唐初立宜州，后为华原县，后又为耀州。宋为感义军，又改感德军，又为耀州如故。金因之。元至元元年，并华原县入州，又并美原入富平。领三县：

16. <span id="志第十二_地理三-陕西诸道行御史台-16"></span>
三原，下。 富平，下。 同官。下。

17. <span id="志第十二_地理三-陕西诸道行御史台-17"></span>
乾州，下。 唐以高宗乾陵所在，改醴泉县为奉天，又升为乾州。宋改醴州。

18. <span id="志第十二_地理三-陕西诸道行御史台-18"></span>
金复改乾州。元至元元年，并奉天县入州。五年，复置奉天，省好畤入焉，又割永寿来属，后又改奉天为醴泉。领三县：

19. <span id="志第十二_地理三-陕西诸道行御史台-19"></span>
醴泉，下。 武功，下。 永寿。下。宋、金属邠州。至元十五年，徙县治于麻亭。

20. <span id="志第十二_地理三-陕西诸道行御史台-20"></span>
商州，下。 唐初为商州，又改上洛郡，又复为商州。宋及元皆因之。领一县：

21. <span id="志第十二_地理三-陕西诸道行御史台-21"></span>
洛南。下。

22. <span id="志第十二_地理三-陕西诸道行御史台-22"></span>
延安路，下。 唐初为延州，又改延安郡，又为延州。宋为延安府。金为鄜延路。元改延安路。户六千五百三十九，口九万四千六百四十一。壬子年数。

23. <span id="志第十二_地理三-陕西诸道行御史台-23"></span>
领县八、州三。州领八县。本路屯田四百八十余顷。

24. <span id="志第十二_地理三-陕西诸道行御史台-24"></span>
县八

25. <span id="志第十二_地理三-陕西诸道行御史台-25"></span>
肤施，下。 甘泉，下。 宜川，下。元初置司候司。至元六年，省入宜川。 延长，下。 延川，下。 安定，下。本宋旧堡，元壬子年升为安定县。至元元年，析置丹头县。四年，并丹头入本县。 安塞，下。本金旧堡，壬子年升为县。 保安。下。金为保安州，至元六年，降为县。

26. <span id="志第十二_地理三-陕西诸道行御史台-26"></span>
州三

27. <span id="志第十二_地理三-陕西诸道行御史台-27"></span>
鄜州，下。 唐初为鄜州，又改洛交郡，又复为鄜州。宋、金因之。旧领洛交、洛川、鄜城、直罗四县。元至元四年，并鄜城入洛川，又并洛交、直罗入州。

28. <span id="志第十二_地理三-陕西诸道行御史台-28"></span>
六年，废坊州，以中部、宜君二县来属。领三县：

29. <span id="志第十二_地理三-陕西诸道行御史台-29"></span>
洛川，下。 中部，下。 宜君。下。

30. <span id="志第十二_地理三-陕西诸道行御史台-30"></span>
绥德州，下。 唐绥州，又改上郡，又为绥州。宋为绥德军。金为州，领八县。归附后，并嗣武入米脂，绥平入怀宁。至元四年，并定戎入米脂，怀宁入青涧，又并义合、绥德入本州。领二县：

31. <span id="志第十二_地理三-陕西诸道行御史台-31"></span>
青涧，下。 米脂。下。

32. <span id="志第十二_地理三-陕西诸道行御史台-32"></span>
葭州，下。 唐银州。宋为晋宁军。金改为葭州。元至元六年，并通秦、弥川、葭卢入州，并太和入神木，建宁入府谷。领三县：

33. <span id="志第十二_地理三-陕西诸道行御史台-33"></span>
神木，下。元初创立云州于古麟州之神木寨。至元六年，废州为县。 吴堡，下。 府谷。下。后唐为府州。元初建州治。至元六年，废为县。

34. <span id="志第十二_地理三-陕西诸道行御史台-34"></span>
兴元路，下。 唐为梁州，又改汉中郡，又为兴元府。宋仍旧名。元立兴元路总管府，久之，以凤、金、洋三州隶焉。宋时领南郑、西县、褒城、廉水、城固五县，后废廉水入南郑。元初割出西县属沔州，以洋州西乡县来属。户二千一百四十九，口一万九千三百七十八。至元二十七年数。 领县四、州三。

35. <span id="志第十二_地理三-陕西诸道行御史台-35"></span>
县四

36. <span id="志第十二_地理三-陕西诸道行御史台-36"></span>
南郑，下。 城固，下。 褒城，下。 西乡，下。

37. <span id="志第十二_地理三-陕西诸道行御史台-37"></span>
州三

38. <span id="志第十二_地理三-陕西诸道行御史台-38"></span>
凤州，下。 唐初为凤州，后升节度府。宋为团练州。至元五年，以在郭梁泉县并入州，隶兴元路。

39. <span id="志第十二_地理三-陕西诸道行御史台-39"></span>
洋州，下。 唐改洋川郡，又复为洋州，后更革不常。宋复为洋州。元至元二年，省兴道、真符二县入州。

40. <span id="志第十二_地理三-陕西诸道行御史台-40"></span>
金州，下。 唐改西城郡为金州。宋升为金房开达四州路。元为散州。

41. <span id="志第十二_地理三-陕西诸道行御史台-41"></span>
陕西汉中道肃政廉访司

42. <span id="志第十二_地理三-陕西诸道行御史台-42"></span>
凤翔府，唐为扶风郡，又为凤翔府，号西京。宋、金因其名。元初割平凉府、秦、陇、德顺、西宁、镇原州隶巩昌路，废恆州，以所领盩啡县隶安西府路，寻立凤翔路总管府。至元九年，更为散府。户二千八十一，口一万四千九百八。壬子年数。 领县五：

43. <span id="志第十二_地理三-陕西诸道行御史台-43"></span>
凤翔，下。屯田九十顷有奇。 扶风，下。 岐山，下。 宝鸡，下。

44. <span id="志第十二_地理三-陕西诸道行御史台-44"></span>
麟游。下。

45. <span id="志第十二_地理三-陕西诸道行御史台-45"></span>
邠州，下。 唐豳州，以字类幽，改为邠。宋、金以来皆因之。领县二：

46. <span id="志第十二_地理三-陕西诸道行御史台-46"></span>
新平，下。 淳化。下。至元七年，并三水入本县。

47. <span id="志第十二_地理三-陕西诸道行御史台-47"></span>
泾州，下。 唐改安定郡，后仍为泾州。宋改彰化军。旧领保定、长武、灵台、良原四县。金改保定县为泾川。元初以隶都元帅府，立总司辖邠州，后属巩昌都总帅府，或隶平凉府、陕西省，所隶不一，今直隶省。领县二：

48. <span id="志第十二_地理三-陕西诸道行御史台-48"></span>
泾川，下。泾州治此，即保定。 灵台。下。至元七年，并归泾川。十一年复立，以良原并入，而长武仍并于泾川。

49. <span id="志第十二_地理三-陕西诸道行御史台-49"></span>
开成州，下。 唐原州。宋为镇戎军。金升镇戎州。元初仍为原州。至元十年，皇子安西王分治秦、蜀，遂立开成府，仍视上都，号为上路。至治三年，降为州。领县一、州一。

50. <span id="志第十二_地理三-陕西诸道行御史台-50"></span>
县一

51. <span id="志第十二_地理三-陕西诸道行御史台-51"></span>
开成。

52. <span id="志第十二_地理三-陕西诸道行御史台-52"></span>
州一

53. <span id="志第十二_地理三-陕西诸道行御史台-53"></span>
广安州。本镇戎地，金升为县，隶镇戎州，经乱荒废。元至元十年，安西王封守西土，既立开成路，遂改为广安县，募民居止，未几户口繁夥。十五年升为州，仍隶本路。

54. <span id="志第十二_地理三-陕西诸道行御史台-54"></span>
庄浪州。下。沿革阙。成宗大德八年二月，降庄浪路为州。

55. <span id="志第十二_地理三-陕西诸道行御史台-55"></span>
巩昌等处总帅府

56. <span id="志第十二_地理三-陕西诸道行御史台-56"></span>
巩昌府，唐初置渭州，后曰陇西郡，寻陷入吐蕃。宋复得其地，置巩州。金为巩昌府。元初改巩昌路便宜都总帅府，统巩昌、平凉、临洮、庆阳、隆庆五府及秦、陇、会、环、金、德顺、徽、金洋、安西、河、洮、岷、利、巴、沔、龙、大安、褒、泾、邠、宁、定西、镇原、阶、成、西和、兰二十七州，又于成州行金洋州事。

57. <span id="志第十二_地理三-陕西诸道行御史台-57"></span>
至元五年，割安西州属脱思麻路总管府。六年，以河州属吐蕃宣慰司都元帅府。七年，并洮州入安西州。八年，割岷州属脱思麻路。十三年，立巩昌路总管府。十四年，复行便宜都总帅府事，其年割隆庆府，利、巴、大安、褒、沔、龙等州隶广元路。二十一年，又以泾、邠二州隶陕西汉中道宣慰司，而帅府所统者，巩昌、平凉、临洮、庆阳，府凡四；秦、陇、宁、定西、镇原、阶、成、西和、兰、会、环、金、德顺、徽、金洋，州凡十有五。户四万五千一百三十五，口三十六万九千二百七十二。壬子年数。 领司一、县五。

58. <span id="志第十二_地理三-陕西诸道行御史台-58"></span>
录事司。

59. <span id="志第十二_地理三-陕西诸道行御史台-59"></span>
县五

60. <span id="志第十二_地理三-陕西诸道行御史台-60"></span>
陇西，下。 宁远，下。 伏羌，下。本旧寨，至元十三年升县。 通渭，下。 鄣县。下。宋名盐川寨，金为镇，至元十七年，置今县。

61. <span id="志第十二_地理三-陕西诸道行御史台-61"></span>
平凉府，唐为马监，隶原州。宋为泾原路，升平凉军。金立平凉府。元初并潘原县入平凉，化平入华亭，隶巩昌帅府。领县三：

62. <span id="志第十二_地理三-陕西诸道行御史台-62"></span>
平凉，下。屯田一百一十五顷。 崇信，下。 华亭。下。

63. <span id="志第十二_地理三-陕西诸道行御史台-63"></span>
临洮府，唐临洮军。宋为镇洮军，又为熙州。金为临洮府。元至元十三年，复以渭源堡升为县。领县二：

64. <span id="志第十二_地理三-陕西诸道行御史台-64"></span>
狄道，下。 渭源，下。

65. <span id="志第十二_地理三-陕西诸道行御史台-65"></span>
庆阳府，唐庆州。宋环庆路，改庆阳军，又升府。金为庆原路。元初改为庆阳散府，至元七年，并安化、彭原入焉。领县一：

66. <span id="志第十二_地理三-陕西诸道行御史台-66"></span>
合水。下。

67. <span id="志第十二_地理三-陕西诸道行御史台-67"></span>
秦州，中。 唐初为秦州。宋为天水郡。金为秦州。旧领六县。元至元七年，并鸡川、陇城入秦安，治坊入清水。领县三：

68. <span id="志第十二_地理三-陕西诸道行御史台-68"></span>
成纪，中。 清水，中。 秦安。下。

69. <span id="志第十二_地理三-陕西诸道行御史台-69"></span>
陇州，中。 唐改汧阳郡，复为陇州。宋、金置防御使。旧领四县。元至元七年，省吴山、陇安入汧源，十三年，罢防御使为散郡。有吴山为西镇。 领县二：

70. <span id="志第十二_地理三-陕西诸道行御史台-70"></span>
汧源，中。 汧阳。下。

71. <span id="志第十二_地理三-陕西诸道行御史台-71"></span>
宁州，下。 唐初改北地郡为宁州。宋、金因之。元至元七年，并襄乐、安定、定平入州。领县一：

72. <span id="志第十二_地理三-陕西诸道行御史台-72"></span>
真宁。下。

73. <span id="志第十二_地理三-陕西诸道行御史台-73"></span>
定西州，下。 本唐渭州西市，五代沦于先零。宋置定西城。金改定西县，复升为州，仍置安西县，倚郭，通西二寨，并置县来属。元至元三年，并三县入本州。屯田四百六十七顷。

74. <span id="志第十二_地理三-陕西诸道行御史台-74"></span>
镇原州，下。 唐原州，又为平凉郡。宋、金因之。元改镇原州，以镇戎州之东山、三川二县来属。至元七年，例并州县，遂以临泾、彭阳及东山、三川四县入本州。屯田四百二十六顷有奇。

75. <span id="志第十二_地理三-陕西诸道行御史台-75"></span>
西和州，下。 唐岷州，又改和政郡，又仍为岷州。宋改曰西和。旧领县三，大潭、祐川军兴久废，惟有长道一县，元至元七年，亦并入本州。

76. <span id="志第十二_地理三-陕西诸道行御史台-76"></span>
环州，下。 唐改威州。宋复为环州，后与庆州定为环庆路。金隶庆阳府。

268. <span id="志第十二_地理三-陕西诸道行御史台-268"></span>
河西陇北道肃政廉访司

269. <span id="志第十二_地理三-陕西诸道行御史台-269"></span>
甘州路，上。 唐为甘州，又为张掖郡，宋初为西夏所据，改镇夷郡，又立宣化府。元初仍称甘州。至元元年，置甘肃路总管府。八年，改甘州路总管府。十八年，立行中书省，以控制河西诸郡。户一千五百五十，口二万三千九百八十七。

270. <span id="志第十二_地理三-陕西诸道行御史台-270"></span>
至元二十七年数。本路黑山、满峪、泉水渠、鸭子翅等处屯田，计一千一百六十余顷。

271. <span id="志第十二_地理三-陕西诸道行御史台-271"></span>
永昌路，下。 唐凉州。宋初为西凉府，景德中陷入西夏。元初仍为西凉府。

272. <span id="志第十二_地理三-陕西诸道行御史台-272"></span>
至元十五年，以永昌王宫殿所在，立永昌路，降西凉府为州隶焉。

273. <span id="志第十二_地理三-陕西诸道行御史台-273"></span>
西凉州。下。

274. <span id="志第十二_地理三-陕西诸道行御史台-274"></span>
肃州路，下。 唐为肃州，又为酒泉郡。宋初为西夏所据。元太祖二十一年，西征，攻肃州下之。世祖至元七年，置肃州路总管府。户一千二百六十二，口八千六百七十九。至元二十七年数。

275. <span id="志第十二_地理三-陕西诸道行御史台-275"></span>
沙州路，下。 唐为沙州，又为敦煌郡。宋仍为沙州，景祐初，西夏陷瓜、沙、肃三州，尽得河西故地。金因之。元太祖二十二年，破其城以隶八都大王。至元十四年，复立州。十七年，升为沙州路总管府，瓜州隶焉。沙州去肃州千五百里，内附贫民欲乞粮沙州，必须白之肃州，然后给与，朝廷以其不便，故升沙州为路。

276. <span id="志第十二_地理三-陕西诸道行御史台-276"></span>
瓜州，下。 唐改为晋昌郡，复为瓜州。宋初陷于西夏。夏亡，州废。元至元十四年复立。二十八年徙居民于肃州，但名存而已。

277. <span id="志第十二_地理三-陕西诸道行御史台-277"></span>
亦集乃路，下。 在甘州北一千五百里，城东北有大泽，西北俱接沙碛，乃汉之西海郡居延故城，夏国尝立威福军。元太祖二十一年内附。至元二十三年，立总管府。二十三年，亦集乃总管忽都鲁言：“所部有田可以耕作，乞以新军二百人凿合即渠于亦集乃地，并以傍近民西僧余户助其力。”从之。计屯田九十余顷。

278. <span id="志第十二_地理三-陕西诸道行御史台-278"></span>
宁夏府路，下。 唐属灵州。宋初废为镇，领番部。自唐末有拓拔思恭者镇夏州，世有银、夏、绥、宥、静五州之地。宋天禧间，传至其孙德明，城怀远镇为兴州以居，后升兴庆府，又改中兴府。元至元二十五年，置宁夏路总管府。至元八年，立西夏中兴等路行尚书省。元贞元年，革宁夏路行中书省，并其事于甘肃行省。

279. <span id="志第十二_地理三-陕西诸道行御史台-279"></span>
领州三。本路枣园、纳怜站等处屯田一千八百顷。

280. <span id="志第十二_地理三-陕西诸道行御史台-280"></span>
灵州，下。 唐为灵州，又为灵武郡。宋初陷于夏国，改为翔庆军。

281. <span id="志第十二_地理三-陕西诸道行御史台-281"></span>
鸣沙州，下。 隋置环州，立鸣沙县。唐革州以县隶灵州。宋没于夏国，仍旧名。元初立鸣沙州。屯田四百四十余顷。

282. <span id="志第十二_地理三-陕西诸道行御史台-282"></span>
应理州，下。 与兰州接境，东阻大河，西据沙山。考之图志，乃唐灵武郡地。其州城未详建立之始，元初仍立州。

283. <span id="志第十二_地理三-陕西诸道行御史台-283"></span>
山丹州，下。 唐为删丹县，隶甘州。宋初为夏国所有，置甘肃军。元初为阿只吉大王分地。至元六年，行山丹城事，删讹为山。二十二年，升为州，隶甘肃行省。

284. <span id="志第十二_地理三-陕西诸道行御史台-284"></span>
西宁州，下。 唐置鄯州，理湟水县，上元间没于土蕃，号青唐城。宋改为西宁州。元初为章吉驸马分地。至元二十三年，立西宁州等处拘榷课程所。二十四年，封章吉为宁濮郡王，以镇其地。

285. <span id="志第十二_地理三-陕西诸道行御史台-285"></span>
兀剌海路。阙。太祖四年，由黑水城北兀剌海西关口入河西，获西夏将高令公，克兀剌海城。
